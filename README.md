# 🚀  Multi-Modal AI Assistant (Text + Voice + Vision)

### Idea

Build an assistant that:

* Understands **speech (ASR)**
* Processes **text (NLP)**
* Sees environment (CV)
* Responds intelligently

### Stack

* NLP: Hugging Face Transformers
* Speech: Whisper
* Vision: OpenCV

### Features

* “What is in front of me?”
* “Summarize what I said”
* Context memory
